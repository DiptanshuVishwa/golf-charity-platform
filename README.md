# 🚀 Golf Charity Subscription Platform

A full-stack web application that combines performance tracking, reward-based draws, and charity contributions.

## 🌐 Live Demo
- Frontend: https://golf-charity-platform-eight-rho.vercel.app
- Backend: https://golf-charity-platform-0prl.onrender.com

---

## ✨ Features

### 👤 User Features
- Secure Authentication (JWT)
- Dashboard with real-time data
- Add & manage golf scores (rolling 5 logic)
- Subscription system (monthly/yearly)
- Charity selection & contribution tracking

### 🎯 System Features
- Random draw engine
- Match calculation (3, 4, 5 matches)
- Prize logic simulation

### 👑 Admin Features
- View all users
- Run monthly draw
- View winners

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)

### Backend
- Node.js
- Express.js
